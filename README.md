# Система управления рецептами

Консольное приложение для управления кулинарными рецептами. Позволяет создавать блюда, добавлять ингредиенты, масштабировать порции под нужное количество человек и автоматически генерировать итоговый список покупок.

## Установка

Для проверки проекта клонируйте репозиторий и установите зависимости:

```bash
git clone https://github.com/vspiradov/hw2-programming-tech.git
cd hw2_recipes
pip install -r requirements.txt
```

## Использование (Тестирование)

Проект покрыт тестами с использованием библиотеки `pytest`. Для запуска тестов выполните в корневой папке проекта команду:

```bash
pytest
```

Для получения подробного отчета о прохождении каждого отдельного теста используйте флаг `-v`:

```bash
pytest -v
```
